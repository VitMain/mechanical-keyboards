# Dactyl Manuform 5x6 - Honey & Milk, DE Layout

## Install

Create a link from this keymap folder into `keyboards/handwired/dactyl_manuform/5x6/keymaps/`.


## Compile

    qmk compile -kb handwired/dactyl_manuform/5x6 -km honey-milk-de
